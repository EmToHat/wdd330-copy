Local Storage

Writing an application means that you are communicating with a user.

True communication should be bidirectional. The user should be able to communiate to you.

The localStorage object allows you to save key/value pairs in the browser.

Syntax:
localStorage.setItem("lastname", "Smith");
localStorage.getItem("lastname");

localStorage object will store data w/no expiration date. The data stored won't delete when the browser is closed.

If you want to remove data from localStorage, you use removeItem(key). If you want to clear the local storage you would use, localStorage.clear();
