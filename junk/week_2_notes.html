Sets

Sets were introduced to the specification in ES6. A set is a data structure that represents a collection of unique values, so it cannot include any duplicate values. They are similar in concept to a mathematical set, although (for the time being at least) they don't contain mathematical set operations such as union, intersection and product.
Sets offer a useful way to keep track of data without having to check if any values have been duplicated. It's also quick and easy to check if a particular value is in a set, which can be a slow operation if an array is used.

Loops

Loops will repeat a piece of code over and over again according to certain conditions.

// While Loop, this loop repeatedly runs a block of code while a certain condition remains true.
// This loop will be infinite unless a condition is met at some point. The code could be stuck if there isn't a stopping condition.
while (condition) {
// code to be executed
}

// For loops are the most common loop in JS.
for (initialization ; condition ; after) {
do something
}

Functions CH 04

Defining a Function

// To define a function literal, we use the function declaration
function hello() {
console.log('Hello World!');
}

// Another way of defining a function literal is to create a function expression. This function is an anonymous function because it doesn't have a name.
const goodbye = function(){
console.log('Goodbye World!');
};

// But you can create a named function also
const goodbye = function bye(){
console.log('Goodbye World!');
};

// A function can also be declared using constructor Function(). The body of this function is entered as a string.
const hi = new Function('console.log("Hi World!");');

Invoking a Function

Invoking the function will run the code inside the function's body. In order to invoke the function you enter it's name followed by parentheses.
The function can be invoked over and over again just by typing its name followed by parentheses.

hello();
<< 'Hello World!'

Return Values

All function RETURN a value, this can be specified when using the return statement. Any function that doesn't specifically say return will return undefined by default.

function howdy() {
return 'Howdy World!';
}

Parameters & Arguments

Parameters and arguments are terms that are often used interchangeably to represent values provided for the function as an input.
These two terms do have a difference but it is subtle. Any parameters a function needs are set when the function is defined. But when the function is invoked, the function is provided with arguments.

Callbacks

Functions can be given as a parameter to other functions. When a function is passed as an argument to another function this is known as a callback.

function sing(song) {
console.log(`I'm singing along to ${song}`);
}
sing('Let It Go')
<< 'I'm singing along to Let It Go'

function sing(song, callback) {
console.log(`I'm singing along to ${song}.`);
callback();
}
