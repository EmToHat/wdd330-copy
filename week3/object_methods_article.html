Object Methods Article

Object Methods, "this"

Objects are usually created to represent entities of the real world, like users, orders and so on.

let user = {
name: "John",
age: 30
};

In the real world, a user can act: select something from the shopping cart, login, logout etc.

Method Examples

Teach the user to say hello

let user = {
name: "John",
age: 30
};
// function expression
user.sayHi = function() {
alert("Hello!")
};
user.sayHi(); // Output = Hello!
